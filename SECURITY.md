# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | ✅ Active support  |
| Older   | ❌ No support      |

Only the latest release of AxCom receives security updates.

## Reporting a Vulnerability

**⚠️ Do NOT report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in AxCom, please report it
responsibly by contacting us directly:

**Email:** security@axiolon.com

### What to Include

When reporting a vulnerability, please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any proof-of-concept code, if available
- The version of the Software affected
- Your contact information for follow-up

### Response Timeline

| Stage                        | Timeframe           |
| ---------------------------- | ------------------- |
| Acknowledgement of report    | Within 48 hours     |
| Initial assessment           | Within 5 business days |
| Status update                | Within 10 business days |
| Resolution (if applicable)   | Depends on severity |

### Responsible Disclosure

We kindly request that you:

- Allow us reasonable time to address the vulnerability before any public disclosure
- Avoid exploiting the vulnerability beyond what is necessary to demonstrate the issue
- Do not access, modify, or delete data belonging to other users

### Recognition

We value the security research community. Reporters of valid vulnerabilities may be
acknowledged (with permission) in our release notes, unless they prefer to remain anonymous.

## Scope

This security policy covers the core AxCom software distributed by Axiolon Labs.
Third-party integrations, plugins, or modifications made by licensees are outside the scope
of this policy.

## Legal

Security research conducted in good faith and in accordance with this policy will not be
subject to legal action by Axiolon Labs. We consider security research conducted consistent
with this policy to be authorized and will not pursue civil or criminal action.
